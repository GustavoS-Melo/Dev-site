# Gustavo Melo — Portfólio Profissional

Este projeto corresponde ao meu site pessoal e portfólio profissional como desenvolvedor front-end.  
O objetivo do site é apresentar meus projetos, habilidades e informações de contato de forma clara, direta e profissional.

Trata-se de um site real, desenvolvido e publicado para uso próprio, atualmente em produção.

## Sobre o projeto
O portfólio foi desenvolvido com foco em simplicidade, boa organização visual e experiência do usuário. Ele funciona como um ponto central para apresentação do meu trabalho, reunindo projetos reais, estudos e informações relevantes sobre minha atuação como desenvolvedor.

Todo o processo, desde o desenvolvimento até o deploy, foi realizado por mim.

## Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript
- Git e GitHub

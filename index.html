Habilidades

HTML

Estruturação semântica e otimizada para SEO.

HTML (HyperText Markup Language) é o esqueleto da página, responsável por organizar e definir os elementos que aparecem na tela, servindo como base para o CSS (estilo) e o JavaScript (funcionalidade).

CSS

Layouts modernos, responsivos e animados.

CSS (Cascading Style Sheets) é usado para definir o visual e o estilo de uma página web. Enquanto o HTML estrutura o conteúdo, o CSS controla cores, tamanhos, fontes, espaçamentos, layouts, animações e efeitos visuais.

JavaScript

Interatividade, lógica e manipulação dinâmica.

JavaScript é a linguagem de programação responsável por trazer interatividade e dinamismo às páginas web. Com ele, elementos da página podem responder a ações do usuário, como cliques, digitação e movimentos do mouse, além de permitir animações, validação de formulários, requisições a APIs, atualização de conteúdo sem recarregar a página e muito mais.

React

Interfaces componentizadas e performáticas.

React é uma biblioteca JavaScript usada para criar interfaces de usuário de forma rápida, organizada e eficiente. Ele funciona com componentes, que são blocos reutilizáveis de código que representam partes da interface como botões, cards, menus, formulários etc.

Node.js

APIs rápidas e seguras.

Node.js é um ambiente de execução JavaScript no lado do servidor. Enquanto o JavaScript tradicional roda apenas no navegador, o Node.js permite usar JavaScript para criar back-ends, APIs, servidores e aplicações completas fora do navegador.

UI / UX

Design intuitivo e focado em experiência.

UX é sobre como o usuário se sente ao usar um produto. Trabalha com pesquisas, testes, fluxos, arquitetura de informação e usabilidade para garantir que tudo seja simples, intuitivo e agradável. UI é sobre a aparência e a interação visual. Define cores, tipografias, botões, espaçamentos, ícones e todos os elementos gráficos que o usuário vê e toca.
